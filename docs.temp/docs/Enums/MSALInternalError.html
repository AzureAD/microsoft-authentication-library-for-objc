MSALInternalError

enum MSALInternalError {}

MSALInternalError enum contains all possible errors under MSALInternalErrorCodeKey. This enum exists only for the reference, you should not try to handle these errors in runtime.

  • MSALInternalErrorInvalidParameter

    A required parameter was not provided, or a passed in parameter was invalid. See MSALErrorDescriptionKey for more information.

    Declaration

    Objective-C

    MSALInternalErrorInvalidParameter = -42000

    Swift

    case invalidParameter = -42000
  • MSALInternalErrorRedirectSchemeNotRegistered

    The required MSAL URL scheme is not registered in the app’s info.plist. The scheme should be msal

    e.g. an app with the client ID abcde-12345-vwxyz-67890 would need to register the scheme msalabcde-12345-vwxyz-67890 and add the following to the info.plist file:

    CFBundleURLTypes CFBundleTypeRole Editor CFBundleURLName $(PRODUCT_BUNDLE_IDENTIFIER) CFBundleURLSchemes msalabcde-12345-vwxyz-67890
